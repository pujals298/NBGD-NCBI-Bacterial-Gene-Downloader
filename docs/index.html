Objetivo del Proyecto

Este pipeline automatiza el proceso de análisis filogenético para genes de cualquier organismo desde NCBI RefSeq, permitiendo estudios de diversidad genética y evolución.

¿Qué es la filogenia?

La filogenia es el estudio de las relaciones evolutivas entre organismos. Los árboles filogenéticos muestran cómo diferentes cepas o especies comparten ancestros comunes, permitiendo entender la evolución y propagación de genes.

¿Para qué sirve?

  • Analizar la diversidad genética de cualquier organismo
  • Identificar patrones de evolución de genes
  • Comparar secuencias de genes específicos entre cepas o especies
  • Generar árboles filogenéticos para publicaciones científicas

Resumen del Pipeline

El pipeline consiste en dos pasos consecutivos que transforman datos crudos de NCBI en análisis filogenéticos listos para interpretar.

Concepto: Pipeline

Un pipeline es una serie de programas que se ejecutan en secuencia, donde la salida de cada paso es la entrada del siguiente. Esto automatiza tareas repetitivas y garantiza consistencia en los resultados.

Cómo Funciona

Paso 1: Búsqueda y Extracción de Genes

refseq.py

Este script se conecta a la base de datos NCBI RefSeq y:

  1. Busca genomas de cualquier organismo usando TaxID o nombre
  2. Extrae secuencias de genes específicos (por nombre de gen o producto)
  3. Guarda los resultados en formato FASTA
  4. Genera un archivo Excel con información de control de calidad

Paso 2: Alineamiento, Poda y Árbol Filogenético

aln_trim_tree.py

Este script procesa las secuencias descargadas:

  1. Alinea las secuencias usando MUSCLE (alineamiento múltiple)
  2. Poda las regiones problemáticas usando ClipKIT
  3. Construye un árbol filogenético usando IQ-TREE
Herramienta Función Entrada Salida
MUSCLE Alineamiento múltiple de secuencias FASTA (secuencias no alineadas) FASTA (alineado)
ClipKIT Poda de alineamientos FASTA alineado FASTA podado
IQ-TREE Construcción de árboles filogenéticos FASTA podado Árbol (.treefile), estadísticas

Inicio Rápido

Sigue estos pasos para ejecutar el pipeline en tu computadora.

1

Configurar el entorno

# Linux/Mac
bash ./setting.sh

# Windows
powershell -ExecutionPolicy Bypass -File .\setting.ps1

Esto crea un entorno virtual de Python e instala las dependencias necesarias.

2

Ejecutar el pipeline

python run.py

El script te preguntará qué quieres hacer:

  • 1 → Solo Paso 1 (descargar genes)
  • 2 → Solo Paso 2 (alinear/árbol)
  • 3 → Ambos pasos seguidos
3

Proporcionar los datos requeridos

El pipeline te pedirÃ¡:

  • Email para NCBI (obligatorio)
  • Clave API de NCBI (opcional, pero recomendado)
  • Nombre del gen a buscar (ej: 16S rRNA, cox1)
  • TaxID del organismo a estudiar (ej: 9606 para Homo sapiens)
Consejo

Obtén una clave API de NCBI en tu cuenta de NCBI. Es gratuita y permite descargas más rápidas.

Conceptos Clave

Glosario de términos importantes para entender el pipeline y el análisis filogenético.

RefSeq

Base de datos de referencia de NCBI con secuencias curadas y de alta calidad. Es el "estándar oficial" para análisis genéticos.

FASTA

Formato de texto para almacenar secuencias de ADN/proteínas. Cada secuencia comienza con una línea de descripción seguida de las letras de la secuencia.

CDS (Coding Sequence)

Secuencia de codificación: la parte del ADN que codifica para una proteína. Contiene la información genética real.

Alineamiento

Proceso de organizar secuencias para identificar posiciones homologas (heredadas de un ancestro común). Revela similitudes y diferencias.

Poda (Trimming)

Eliminar columnas dudosas o con muchos huecos del alineamiento para mejorar la calidad del árbol filogenético.

Árbol Filogenético

Diagma que muestra las relaciones evolutivas entre secuencias. Los nodos representan ancestros comunes y las ramas muestran divergencia.

Bootstrap

Método estadístico que mide la confianza de cada rama del árbol. Valores altos (>70%) indican ramas fiables.

Entrez

Sistema de búsqueda de NCBI que permite acceder programáticamente a sus bases de datos biológicas.

TaxID

Identificador numérico único para organismos en NCBI. Por ejemplo, Homo sapiens = 9606.

Homología

Secuencias que descienden de un ancestro común. Puede ser ortología (especies diferentes) o paralogía (mismo genoma).

Estructura del Proyecto

 ncbi-beginner-downloader/
├── run.py                          # Punto de entrada del pipeline
├── setting.sh / setting.ps1       # Scripts de configuración
├── requirements.txt                # Dependencias de Python
├── 1 SEARCH GENES/
│   └── refseq.py                   # Paso 1: Descarga de NCBI
└── 2 ALN_TRIM_TREE/
    ├── aln_trim_tree.py            # Paso 2: Alineamiento y árbol
    ├── muscle-win64.v5.3.exe       # MUSCLE para Windows
    └── bin/iqtree2.exe             # IQ-TREE para Windows

Dependencias

  • Python 3.12+ - Intérprete
  • Biopython - Manipulación de secuencias biológicas
  • OpenPyXL - Generación de archivos Excel
  • Certifi - Certificados SSL para NCBI
  • ClipKIT - Poda de alineamientos

Herramientas Externas

  • MUSCLE - Alinear secuencias (instalar con sudo apt install muscle en Linux)
  • IQ-TREE - Construir árboles (instalar con sudo apt install iqtree en Linux)

Argumentos de Línea de Comandos

aln_trim_tree.py

python aln_trim_tree.py --input input.fasta [opciones]

--input FILE         Archivo FASTA de entrada (requerido)
--aligned FILE      Archivo de salida alineado (opcional)
--trimmed FILE      Archivo de salida podado (opcional)
--clipkit-mode MODE Modo de poda: smart-gap, gappy, kpic (default: smart-gap)
--outgroup NAME     Nombre del outgroup para el árbol (opcional)

Formatos de Salida

Paso 1 - Archivos generados:

  • .fasta - Secuencias extraídas
  • _code.fasta - Versión con códigos (si se solicita)
  • .xlsx - Control de calidad en Excel

Paso 2 - Archivos generados:

  • _aligned.fasta - Secuencias alineadas
  • _aligned_trimmed.fasta - Alineamiento podado
  • iqtree_results/ - Directorio con:
    • .treefile - Árbol filogenético (formato Newick)
    • .iqtree - Archivo con estadísticas
